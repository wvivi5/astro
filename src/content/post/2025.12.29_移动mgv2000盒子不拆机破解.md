第一步，盒子通电状态下，按遥控器上的设置按钮，进入设置，登陆密码是10086

然后找到蓝牙

手机或者电脑上下载一个当贝市场或者当贝桌面，通过蓝牙分享给盒子。

按菜单键查看已接收的文件，点击安装，完成后

安装完成后一定要选择打开

顺利进入到当贝市场

在当贝市场找到当贝桌面，安装并运行

找到系统应用，并且找一个叫adbconsole系统应用，打开adb 调试

点击打开

之后就在电脑上操作了

输入adb connect <盒子IP地址>

输入adb devices 显示的结果能看到盒子+ device就显示连接正常

输入adb shell pm list package 查看已经安装的应用（最后等号后面的就是安装包的名字）

输adb shell pm block <包名> 禁用并卸载程序，主要是卸载IPTV相关的登陆应用

输adb shell ifconfig eth0 down/up 停用 | 启用有线（不一定需要）

关于会一直弹出登陆失败的界面，可以卸载以下几个软件包

C:UsersAdministrator>adb shell pm block com.um.auth

C:UsersAdministrator>adb shell pm block com.unionman.gettoken

C:UsersAdministrator>adb shell pm block cn.gd.snm.snmcm

C:UsersAdministrator>adb shell pm unblock cn.gd.snm.snmcm

C:UsersAdministrator>adb shell pm block cn.gd.snm.snmcm

至此，破解完了以后的盒子就和普通的智能盒子一样了

如果想变回原来的iptv盒子，在输入adb shell pm block <包名> 这一步把block改为unblock,其他不变就可以了。

其他盒子破解也可以试一下这个方法。优点是，不用拆机，不用使用TTL等复杂的刷机流程。
