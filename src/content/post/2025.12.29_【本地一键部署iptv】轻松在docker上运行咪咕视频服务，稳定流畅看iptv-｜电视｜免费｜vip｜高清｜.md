 本地一键部署咪咕

轻松在 Docker 上运行本地版咪咕视频服务





\--------------------------------------------------------------------------------

 部署说明



将 localhost 改成你设备的 IP 地址 即可一键部署。

例如：http://192.168.1.10:3000

\--------------------------------------------------------------------------------

 一键部署命令



docker run -d --restart=always \

\--network host \

\-e mport=3000 \

\-e mhost="http://IP:3000" \

\--name migu_video \

  develop767/migu_video

\--------------------------------------------------------------------------------

 进阶操作



 查看日志



docker logs -f migu_video

 更新镜像



docker pull develop767/migu_video:latest

docker rm -f migu_video

 停止 / 删除容器



docker rm -f migu_video

 删除旧镜像



docker rmi develop767/migu_video:latest

 访问示例

部署完成后，在浏览器中访问：

 http://你的设备IP:3000





示例：

http://192.168.1.10:3000

或 http://服务器公网IP:3000

 如果打不开，请检查：



\-  防火墙是否放行端口 3000

\-  宿主系统是否支持 --network host（Linux 推荐）

\-  查看日志排查端口占用或权限问题

 进阶部署（登录vip账号1080p）



docker run -d --restart=always \

\--network host \

\-e mport=3000 \

\-e mhost="http://IP:3000" \

\-e muserId="你的用户ID" \

\-e mtoken="你的Token" \

\-e mrateType=3 \

\--name migu_video \

  develop767/migu_video

 说明



muserId / mtoken 可选

不填也能启动，但 VIP/蓝光内容无法播放

mrateType

2=标清，3=高清，4=蓝光(VIP)

host 模式

直接使用本机网络，局域网内设备可访问

mport / mhost

mport：容器内部端口

mhost：访问地址（用你的设备 IP 和端口）
