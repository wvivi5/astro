适用于 Debian 的详细步骤，使用 Docker 官方源来安装 Docker：



1. \*\*更新现有的软件包列表\*\*：



\    \`\``bash

\    sudo apt-get update

\    \`\``



2. \*\*安装必要的包\*\*：



\    \`\``bash

\    sudo apt-get install -y apt-transport-https ca-certificates curl gnupg lsb-release

\    \`\``



3. \*\*添加 Docker 的官方 GPG 密钥\*\*：



\    \`\``bash

\    curl -fsSL https://download.docker.com/linux/debian/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

\    \`\``



4. \*\*设置稳定的 Docker 仓库\*\*：



\    \`\``bash

\    echo "deb \[arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/debian $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

\    \`\``



5. \*\*更新包索引\*\*：



\    \`\``bash

\    sudo apt-get update

\    \`\``



6. \*\*安装 Docker 引擎\*\*：



\    \`\``bash

\    sudo apt-get install -y docker-ce docker-ce-cli containerd.io

\    \`\``



7. \*\*启动 Docker 并让它在启动时自动运行\*\*：



\    \`\``bash

\    sudo systemctl start docker

\    sudo systemctl enable docker

\    \`\``



8. \*\*验证 Docker 安装是否成功\*\*：



\    \`\``bash

\    sudo docker run hello-world

\    \`\``



如果你想在国内环境中加速 Docker 镜像的下载，可以配置国内镜像加速器。例如，使用阿里云的加速器：



1. \*\*创建或编辑 Docker 的配置文件\*\*：



\    \`\``bash

\    sudo mkdir -p /etc/docker

\    sudo nano /etc/docker/daemon.json

\    \`\``



2. \*\*在配置文件中添加如下内容\*\*：



\    \`\``json

\    {

\    "registry-mirrors": \["https://<your-mirror-URL>"]

\    }

\    \`\``



   例如，使用阿里云的镜像加速器：



\    \`\``json

\    {

\    "registry-mirrors": \["https://<your-alibaba-cloud-mirror-URL>"]

\    }

\    \`\``



3. \*\*重启 Docker 服务\*\*：



\    \`\``bash

\    sudo systemctl daemon-reload

\    sudo systemctl restart docker

\    \`\``



这样，你就可以成功安装 Docker 并使用国内镜像加速器来加速 Docker 镜像的下载。
