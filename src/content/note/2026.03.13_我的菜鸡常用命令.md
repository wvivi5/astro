下面给你一份“够用就行”的 SSH 常用命令清单（偏 Ubuntu/Linux 服务器）。你可以先收藏，平时就靠这些。



───



1) 看我在哪、看目录/文件



pwd                # 当前路径

ls                 # 列出文件

ls -lah            # 详细+人类可读大小+隐藏文件

cd /path/to/dir    # 进入目录

cd ..              # 上一级

cd ~               # 回家目录



2) 查看文件内容（安全，不会改）



cat file.txt               # 直接输出

less file.txt              # 翻页查看（q 退出）

tail -n 50 file.txt        # 看最后 50 行

tail -f file.txt           # 实时追踪日志（Ctrl+C 退出）

head -n 30 file.txt        # 看前 30 行



3) 搜索（非常常用）



grep -n "关键词" file.txt                 # 在文件里找

grep -RIn "关键词" /some/dir              # 目录递归搜索（I忽略二进制）

find . -name "linshi.txt"                 # 按文件名找

find . -type f -mtime -1                  # 找 1 天内改动过的文件



4) 复制/移动/删除（谨慎）



cp a.txt b.txt            # 复制

cp -r dir1 dir2           # 复制目录

mv old new                # 重命名/移动



rm -i file.txt            # 删除（会确认，推荐）

rm -r dir                 # 删除目录（危险，别随便用）



5) 创建/编辑文件（新手推荐顺序）



A. 最简单：nano（容易上手）



nano file.txt



• 保存：Ctrl+O 回车

• 退出：Ctrl+X



B. 稍高级：vim（强大但学习成本高）



vim file.txt



• 进入编辑：按 i

• 保存退出：按 Esc 输入 :wq 回车

• 不保存退出：Esc → :q! 回车



6) 权限/属主（遇到“Permission denied”时用）



chmod +x script.sh         # 给执行权限

chmod 644 file.txt         # 常见文件权限

chown user:user file.txt   # 改属主（root 才能）



7) 看进程/服务（排查问题必备）



top                        # 看实时资源（q退出）

ps aux | grep nginx        # 查进程

df -h                      # 看磁盘空间

free -h                    # 看内存

uptime                     # 系统运行时间/负载



systemd 服务（Ubuntu 常用）：



systemctl status nginx

systemctl restart nginx

systemctl reload nginx

journalctl -u nginx -n 200 --no-pager



8) 网络/端口（你刚用到）



ip a                       # 看网卡IP

ss -ltnp                   # 看监听端口(TCP)

curl -I http://127.0.0.1:18789/   # 测试HTTP是否通



9) 压缩/打包（传文件常用）



tar -czf backup.tgz folder/       # 打包压缩

tar -xzf backup.tgz               # 解压

zip -r backup.zip folder/         # zip 打包

unzip backup.zip                  # unzip 解压



10) 远程传文件（从你电脑/手机到服务器）



如果你在电脑上用 ssh：



scp local.txt root@服务器IP:/root/        # 上传

scp root@服务器IP:/root/a.txt .          # 下载



（手机端工具一般有“上传/下载文件”的按钮，也可以用 SFTP。）
